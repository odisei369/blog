# Blog

Personal blog — hosted on GitHub Pages at `nidobit.com`.

Built with [Hugo](https://gohugo.io/) and the [Blowfish](https://github.com/nunocoracao/blowfish) theme.

## Analytics

- **Umami** — self-hosted on Proxmox LXC, exposed at `umami.nidobit.com` via Cloudflare Tunnel
- **Firebase** — Firestore views/likes counters with anonymous auth

## Branding

### Duck character
- **Nessa** — current name (used in tls-mtls-part-1)
- **Nestor** — alternative. Nestor was the wise old king in the Odyssey — fun irony for a curious duck that's always asking beginner questions.
